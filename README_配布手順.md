# 配布手順ガイド「にほんへの旅」— 教師用README

## パッケージの中身

```
n5-tabi/
├── index.html            旅マップ(ユニット選択画面・生徒の入口)
├── unit0.html            Unit 0 カタカナ語ブースト
├── unit1.html            Unit 1 すうじとじかん
├── manifest.json         PWA設定(ホーム画面追加でアプリ化)
├── sw.js                 オフライン動作用 Service Worker
├── icon-192.png / 512    アプリアイコン
├── voicevox_generate.py  音声一括生成スクリプト(VOICEVOX用)
├── convert_mp3.py        wav → mp3 変換スクリプト(要ffmpeg)
└── audio/                ★先生が生成して追加するフォルダ
```

## 生徒に届くまでの全手順

### STEP 1: 音声を用意する(推奨・省略しても動く)
1. [VOICEVOX](https://voicevox.hiroshiba.jp/) をインストールして起動
2. `python voicevox_generate.py` → `audio/` に全110語のwavが生成される
3. ★印の語のアクセントを耳で確認、必要ならVOICEVOXのGUIで修正して上書き
4. `python convert_mp3.py` → 軽いmp3に変換(全110語で約1.3MB)。wavは削除してよい

※このステップを飛ばしても教材は端末TTSで動きます。後から `audio/` を追加するだけで順次差し替わります。

### STEP 2: GitHub Pagesで公開する(無料・約10分)
1. [github.com](https://github.com) でアカウント作成(無料)
2. 「New repository」→ 名前は例えば `n5-tabi` → Public で作成
3. 「uploading an existing file」から、このフォルダの中身を **audio/ ごと全部** ドラッグ&ドロップしてCommit
4. リポジトリの Settings → Pages → Branch を `main` にして Save
5. 数分後、`https://あなたのID.github.io/n5-tabi/` が公開URLになる

以後の更新は、ファイルを上げ直すだけで全生徒に反映されます。
**注意:** 教材を更新したら `sw.js` の先頭にある `CACHE = "n5tabi-v1"` の番号を1つ上げてください(v1→v2)。これをしないと生徒の端末に古い版がキャッシュされたまま残ることがあります。

### STEP 3: 生徒に配る
1. クラスLINEに公開URLを投稿
2. 生徒はタップして開くだけ。初回に「ホーム画面に追加」をしてもらうとアプリのように使える
   - iPhone: 共有ボタン → ホーム画面に追加
   - Android: ブラウザメニュー(⋮)→ ホーム画面に追加 / アプリをインストール
3. 一度開けばキャッシュされ、**2回目以降はオフラインでも動く**(音声も再生したものから順にキャッシュされる)

## 通信量・サイズの目安

| 内容 | サイズ |
|---|---|
| HTML+アイコン一式 | 約100 KB |
| 音声 Unit 0+1(110語・mp3) | 約1.3 MB |
| 将来:全20ユニット1,000語 | 約12 MB(スマホ写真3枚分) |

初回アクセスを学校のWi-Fiで済ませれば、生徒のモバイル通信はほぼ消費しません。

## よくあるトラブル

| 症状 | 原因と対処 |
|---|---|
| 音が出ない(iPhone) | 本体横のサイレントスイッチがONだと鳴らない機種設定あり。音量も確認 |
| 音声がロボット読みのまま | `audio/` がアップロードされていない。リポジトリにaudioフォルダがあるか確認 |
| 更新したのに古い画面が出る | `sw.js` のバージョン番号を上げたか確認。生徒側は一度閉じて開き直す |
| 進捗が消えた | 進捗は端末ごとのブラウザ保存。ブラウザの閲覧データ削除や機種変更で消えます(仕様)。シークレットモードでは保存されません |
| Notionに貼りたい | HTML直貼りは動きません。公開URLを埋め込み(/embed)にするかリンクを貼る |

## 今後ユニットを追加するとき

1. `unit2.html` などを同じフォルダに追加
2. `index.html` の `UNITS` 配列で該当ユニットに `file:"unit2.html", key:"n5tabi_u2", open:true` を追記
3. `sw.js` の `CORE` に `"unit2.html"` を追加し、`CACHE` のバージョンを上げる
4. 音声は `voicevox_generate.py` に語リストを追記して生成(`u2_00`〜)

これで旅マップに新しい駅が自動的に開通します。
